The `constants.ts` file below and the Locked Landing Page Copy above are the **single source of truth** for all brand, pricing, stats, and copy. Every asset ‚Äî site, ads, emails, sales scripts, Stripe products, pitch decks ‚Äî MUST pull from these sources. **Consistency rule:** Starter/Pro/Premium with setup + monthly pricing must match across: landing page, Stripe Checkout, cold email sequences, pitch decks, onboarding docs, and ROI calculator defaults. If any asset disagrees with `constants.ts`, that asset has a bug.

### üìä Minimal Event Schema (PostHog)

| **Event Name** | **Properties** | **Trigger** |
| --- | --- | --- |
| `cta_click` | which_cta, page, variant | Any CTA button clicked |
| `lead_form_submit` | form_type (analysis / contact), email, business_type | Response-time analysis or contact form submitted |
| `roi_calc_change` | missed_calls, avg_job_value, calculated_revenue | ROI calculator slider moved |
| `checkout_start` | plan (starter / pro / premium) | Stripe Checkout session created |
| `checkout_complete` | plan, setup_fee, monthly, stripe_session_id | Stripe Checkout succeeded |
| `webhook_success` | type (twilio_voice / twilio_sms / stripe), latency_ms | Webhook processed successfully |
| `webhook_fail` | type, error_code, error_message | Webhook processing failed |
| `page_view` | page, referrer, utm_source, utm_medium | Any page loaded |
| `scroll_depth` | page, depth_percent (25 / 50 / 75 / 100) | User scrolls past threshold |

**Rule:** If an event in this schema is not firing in PostHog after deploy, the feature is not shipped.
